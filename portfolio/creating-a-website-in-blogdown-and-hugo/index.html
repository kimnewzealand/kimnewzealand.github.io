Creating a website in blogdown and Hugo

There are a few tutorials on how to create a website using blogdown and Hugo. This is a summary of of my tips if you are using Windows, github repo and hugo creative portfolio theme.

Once you have installed your theme, do not change any of the theme folder files. You can override these with new files in your blog_down source directory.

This theme does not seem to accept list.html in layout.

I am using Atom text editor to edit markdown and html files.

I am using GitKraken to stage, commit and push files from my kimnewzealand.github.io repo.
