Introduction to the Challenge

Creating a minesweeper game was a challenge in Sprint 5 at Dev Academy.

Minesweeper is a single-player puzzle video game. The objective of the game is to clear a rectangular board containing hidden "mines" or bombs without detonating any of them, with help from clues about the number of neighboring mines in each field. The game originates from the 1960s, and has been written for many computing platforms in use today. It has many variations and offshoots.
This blog describes creating a version of the game using HTML, CSS and JavaScript.

Setting up the Board

The first thing to do is to create a board. I found this example by Matt Soria and liked the styling as it looks like the grey blocked styled minesweeper game..

So in a new CSS file I added a body class to center text, determine font and background color using a new color function, relative padding and a the Space mono font. 

                        body {
                            text-align: center;
                            color: color(black);
                            padding: 1em 1em 1em;
                            background-color: color(gray-light);
                            font-family: 'Space Mono', monospace;
                          }


Next for the board we need a new HTML file containing HTML div elements with ids for the board, scoreboard, minecounter, reset smiley face and grid. To keep this game relatively simple, I will not include drop down menus or player levels.

We need to wrap these elements in a darker gray to show the game board, so we can create a div with the class top, with a further class wrap that will position these div elements. Use inline-block? but it is a legacy option https://developer.mozilla.org/en-US/docs/Web/CSS/display. I am trying inline flex.

Things I learnt in this Challenge

  • The head and header tags are different. The head tag is not displayed but it is a container for metadata (data about the HTML) and is placed between the html tag and the body tag. The header tag is used within the body of the website, represents a container for introductory content .
  • There are variations of CSS such as SCSS. The styling sheet I tried in this project is SCSS which is Syntactically Awesome Style Sheet. A list of color properties and values can be reused using variables 
    $colors: (
                                white: #fff,
                                gray-light: #efefef)
    and a function 
     @function color($color-name) {
                                @return map-get($colors, $color-name);
    and https://stackoverflow.com/questions/46400443/what-is-the-difference-between-css-and-scss }
