Introduction

JavaScript and its relationship to HTML and CSS

Web development is like building a house. The HTML language builds the house, the CSS paints the house and the JavaScript switches on the house.

Control flow and loops, arrays and objects

JavaScript is an object-based language

Control flow statements include if, for, while and are used for loops.

Functions and why they are useful

Functions are like “building blocks” of coding They allow the code inside the function to be called many times without having to rewrite the coed each time.
