Changing the favicon in blogdown

I use the Blackburn theme with blogdown.

After checking their are no copyright restrictions, I changed to the website icon or favicon on this website.

I first converted it from png to ico format using the Favicon Generator then changed the favicon file in the path themes\blackburn\static\img.

Then to rebuild the site I used build_site(local = TRUE), et voila!
